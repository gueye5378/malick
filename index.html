TOCOMPOSTE

Description du projet

Intro :
Tocomposte est un projet imaginé développé et bientôt hébergé, j'ai choisi de faire un projet environnemental basé sur le composte.

Objectif du projet :
Grâce à mon application Web une personne vivant en centre-ville (utilisateur client) pourrait être mis en relation avec une personne possédant une solution de tri (utilisateur propriétaire) de déchets comme un composte. Par le biais d'un abonnement l'utilisateur propriétaire pourrait s'occuper des biodéchets d'un utilisateur client.

Technologies :
Sur ce projet j'ai utilisé principalement Node js comme framework j'ai créé mon api avec Express js et j'ai stocké mes données avec Mysql . Ensuite J'ai utilisé EJS comme moteur de template et de l'HTML CSS pour le rendu côté client.
